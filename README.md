Keywords Generator
========

Automatic keywords generator

Exemple :
--------

    $("#areainput").keywordsgenerator({ attachto: $("#areakeywords"), tag: "b", min: 5, max: 30, words: 200, alertmax: false });

Default options :
--------

    min: 4
    max: 25
    words: 50
    tag: null
    alertmid: 40
    alertmax: true
    attachto: null
    callback: null
